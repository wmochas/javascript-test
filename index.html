Resumen

Es un lenguaje de programación que me permite definir el comportamiento de una página web (HTML define el contenido y CSS la apariencia). También es usado en programas de escritorio y servidores como node.js. También algunas BDD como MongoDB usan este lenguaje. Con javascript puedo cambiar el contenido y atributos HTML así como ocultar o mostrar elementos HTML, también me permite modificar los estilos CSS, por medio de DOM. Por medio de BOM puedo interactuar con el browser.

Try / catch

El control de error es similar en sintaxis a java, usa try/catch/finally. La diferencia que solo arroja un tipo de exception

Cuando existe un error, se lanza una exception con un objeto Error, este objeto tiene la propiedad name, con el valor de esta propiedad podemos identificar el tipo de error; mayor referencia en Error

Para lanzar una exception predefinida usamos throw. Esta puede ser String, a Number, a Boolean ó Object.
